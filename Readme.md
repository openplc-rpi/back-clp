1. Enviroment Setup

    $ git clone https://github.com/openclp-rpi/back-clp.git

    $ cd back-clp

    $ python -m venv env

    $ source env/bin/activate

    $ pip install -r requirements.txt

    Make sure you are using Python 3.

2. App Execution

    $ python app.py

4. Rest's Access

    Use Postman or CURL to test the rests.
